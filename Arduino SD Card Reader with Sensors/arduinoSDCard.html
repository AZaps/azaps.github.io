Arduino SD with Sensors

This project is the basis for my groups' 2016 senior design project. This is a simpler version and is currently missing any sensing input. I took this upon myself to see if I could add extra functionality to our project. Due to the limited amount of memory on the Uno, the SD library functions took a lot of space and limited the amount of extra functionality that was needed. I created a custom library that holds all the SD functions to reduce the amount of limited SRAM.

The goal of this project is read any sensor data, format the data, and save it to the SD card so it can used with a C# program I am currently writing. The program is to read the data from the SD card and provide useful, visual feedback to the user in a graphical form that shows trends over a defined time period.

The code for this project is on GitHub and the C# program is still in its very early stages.
