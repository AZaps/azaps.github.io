RGB Sensor and Controller

This project is one of the more entertaining ones. I have a set of RGB light strips that change color based on the connected controller. The controller only has the ability to select a single color or have it change through the spectrum on a preset loop. Well that's boring unless I wanted to use it strictly for ambient lighting. I wanted color changing based on what was coming off the TV, and to do that I needed to get a sensor that could detect the color spectrum and pass it to the controller and the light strips.

Time to crack it open! I opened up the controller and wired three separate wires to the three controlling transistors and another wire for ground. That was essentially it for the controller, the most amount of work was with the light sensor and Arduino.

As for detecting the color from the TV, I went with a RGB sensor from Sparkfun. This was to be hooked up to the Arduino which pass the correct RGB value to the controller.

The code for the project will be available on GitHub and the light strip will be up shortly.
